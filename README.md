# homebrew-vibe

Homebrew formula for [vibe-clock](https://github.com/dexhunter/vibe-clock).

## Installation

```bash
brew tap dexhunter/vibe
brew install vibe-clock
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `vibe-clock` | Track AI coding agent usage across Claude Code, Codex, and OpenCode |

## Usage

```bash
# Show your AI coding stats
vibe-clock stats

# Schedule automatic tracking (macOS LaunchAgent)
vibe-clock schedule --time 11:00

# Check version
vibe-clock --version
```

For full documentation, see the [vibe-clock repo](https://github.com/dexhunter/vibe-clock).
